집탈출

제작의도

  • 엔데믹 이후 급증한 여행 관심도에 따라 여행의 중요요소인 최신 숙박 시설 정보를 제공하고, 예약까지 이 어질 수 있는 여행 예약 플랫폼을 기획했습니다.

주요사항

  • Vue.js를 이용한 프론트엔드 서버 개발
  • Node.js를 이용한 Restful API 백엔드 서버 개발

개발환경

  • HTML
  • CSS
  • JavaScript
  • Vue.js
  • Node.js
  • MySQL
  • AWS

나의 구현 기능

  • REST API를 이용해 내 정보, 예약 정보 구현
  • 랜덤으로 추천 숙소 정보 불러오기
  • 숙소 정보 제공 및 예약
  • 관리자 권한으로 숙소 등록, 회원 관리와 삭제
  • 내 정보 수정
  • 예약한 숙소 정보 내역 및 모달창 구현
  • 회원 탈퇴

상세 구현 기능

    🔎 메인 페이지
  • REST API를 이용해 숙소 데이터 정제 및 분류
  • 랜덤으로 호텔 목록 배치
  • 상세보기 클릭시 해당하는 디테일 페이지로 이동
  • 하단 카테고리 클릭시 숙소리스트 페이지로 이동
  • 예약 페이지에서 Datepicker를 이용한 달력 api이용
  • 체크인 날짜/ 체크아웃 날짜 구현
    🔎 관리자 페이지
  • 관리자 계정으로 로그인시 숙소 등록 페이지, 회원 및 예약 관리 페이지
  • 숙소 등록시 필수 입력값이 없으면 입력 수정 안내
  • 필수 입력값을 담은 formData를 서버로 전송
