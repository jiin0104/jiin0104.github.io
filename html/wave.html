Wave Cinema

제작의도

  • MBTI와 같이 개성이 중요한 사회에서 단 몇 번의 클릭으로 나와 어울리는 영화,
  • 내가 관심있을만한 영화를 추천해주고 사용자 데이터를 분석하여 취향이 비슷한 사용자들의 추천 영화를 보여줍니다

주요사항

  • Vue.js를 이용한 프론트엔드 서버 개발
  • Node.js를 이용한 Restful API 백엔드 서버 개발

개발환경

  • HTML
  • CSS
  • JavaScript
  • Vue.js
  • Node.js
  • MySQL
  • AWS

나의 구현 기능

  • 메인 페이지, 추천 페이지 구현
  • AXIOS를 활용한 REST API 구현 (영화 정보 , 영화 추천)
  • 추천 받았던 목록 불러오기
  • 연령대, 성별이 비슷한 사용자들이 많이 추천 받은 영화 목록 불러오기
  • 회원가입시 선택한 장르와 비슷한 영화 추천
  • 사용자가 선택한 필터링에 따라 영화 추천
  • 사용자들의 추천을 많이 받은 순서대로 영화 불러오기

상세 구현 기능

    🔎 메인 페이지
  • REST API를 이용해 영화 정보 제공 및 추천
  • 로그인 유무에 따라 swiper 컨텐츠를 다르게 배치
  • 비로그인시:지금 상영중인 영화 / 평점 좋은 영화
  • 로그인시:내가 좋아할만한 영화 / 나와 연령대, 성별이 같은 사람이 많이 추천받은 영화
    🔎 추천 페이지
  • 유저 데이터 기반으로 필터링된 장르의 영화들을 랜덤으로 4개배치
  • 영화 상세 정보 페이지 구현
  • 영화 정보, 리뷰 작성, 리뷰 조회 기능
  • 이전에 추천 받은 전체 리스트 조회 기능
